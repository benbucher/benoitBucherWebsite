Here is the procedure to create a new album

1. In the /src folde: copy and paste an existing file album_XXX.html and rename it with appropriate name. Nothin more to do in the file.

2. In the /src/albums folder:
	- copy a new album_XXX and rename it withi appropriate name
	- update the JSON file inside
	- update the photo files inside

3. Update the content of file photo.html accroding to what has been set in the JSON file for the title and date of the album.

And you're good to go!
